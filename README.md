# Welcome to your New Age of Communication 👋

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

## Naming Convention (React Native + TypeScript)

### Folders

- **kebab-case**
- Example:
  user-profile/
  auth-flow/
  shared-ui/

### TSX Files (React Components / Screens)

- **PascalCase**
- Example:
  LoginScreen.tsx
  IconButton.tsx
  UserAvatar.tsx

### TS Files (Logic / Utils / Services)

- **camelCase**
- Used for non-UI code
- Example:
  apiClient.ts
  authService.ts
  dateUtils.ts

### Index files are always in lowercase
